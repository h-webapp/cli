## Getting Started
### install
```console
    npm i h-application -g
```
### init project
```console
    h-application --name helloword
```
### install dependence
```console
    cd helloworld
    npm install
```
### start server
```console
    npm run start
    //run debug server
    npm run start-debug
```
### build
```console
    //build develop
    gulp buildDev
    //build release
    gulp buildRelease
    //build all
    gulp build
    //clean develop
    gulp cleanDev
    //clean release
    gulp cleanRelease
    //clean all
    gulp clean
```
